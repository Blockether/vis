# Fonts

This directory contains fonts bundled with Vis for self-hosted serving.

## Writer

A version of [IBM Plex Mono](https://github.com/IBM/plex) with increased letter-spacing and tuned weights, designed for long-form writing.

- **Author:** Niki Tonsky
- **Source:** https://github.com/tonsky/font-writer
- **License:** SIL Open Font License 1.1

**Files:**
- `Writer-Regular.ttf` (400)
- `Writer-Medium.ttf` (500)
- `Writer-SemiBold.ttf` (600)
- `Writer-Bold.ttf` (700)
- `Writer-Italic.ttf` (400 italic)
- `Writer-MediumItalic.ttf` (500 italic)
- `Writer-SemiBoldItalic.ttf` (600 italic)
- `Writer-BoldItalic.ttf` (700 italic)

## Fira Code

A monospaced programming font with programming ligatures.

- **Author:** Nikita Tonsky
- **Source:** https://github.com/tonsky/FiraCode
- **License:** SIL Open Font License 1.1

**Files:**
- `FiraCode-Light.ttf` (300)
- `FiraCode-Regular.ttf` (400)
- `FiraCode-Medium.ttf` (500)
- `FiraCode-SemiBold.ttf` (600)
- `FiraCode-Bold.ttf` (700)
