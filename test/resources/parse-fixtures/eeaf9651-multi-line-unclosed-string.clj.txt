## Failure 1: Wrong File Paths — Blind Assumption

### Symptom
`v/ls` and `v/rg` throw `ExceptionInfo: Path not found`.

### Real occurrences

| Attempt | Code | Error |
|---------|------|-------|
| iter 2 | `(def tui-ls (v/ls \"src/com/blockether/vis/tui\"))` | `Path not found: …/src/com/blockether/vis/tui` |
| iter 4 | `(v/rg {:all [\"system_prompt\"] :paths [\"src/com/blockether/vis/internal/db.clj\"]})` | `Path not found: …/db.clj` |

The agent assumed directory layouts and file locations without confirming them first.

### Root cause
Agent guessed paths from conventions instead of probing the actual filesystem.
