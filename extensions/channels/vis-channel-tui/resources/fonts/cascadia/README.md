# Bundled Cascadia fonts

These fonts are redistributed for `vis channels tui --standalone`.

Source: Microsoft Cascadia Code release `v2407.24` / `CascadiaCode-2407.24.zip`.
License: SIL Open Font License 1.1. See `OFL-1.1-Cascadia.txt` in this directory.

Bundled variable TTF variants:

- `CascadiaCode.ttf`
- `CascadiaCodeItalic.ttf`
- `CascadiaCodePL.ttf`
- `CascadiaCodePLItalic.ttf`
- `CascadiaCodeNF.ttf`
- `CascadiaCodeNFItalic.ttf`
- `CascadiaMono.ttf`
- `CascadiaMonoItalic.ttf`
- `CascadiaMonoPL.ttf`
- `CascadiaMonoPLItalic.ttf`
- `CascadiaMonoNF.ttf`
- `CascadiaMonoNFItalic.ttf`

Variant guide:

- Code: programming ligatures; best for source/code display.
- Mono: no ligatures; best for terminal/editor writing where stable character-by-character rendering matters.
- PL: Powerline glyphs.
- NF: Nerd Font glyphs; larger files.
